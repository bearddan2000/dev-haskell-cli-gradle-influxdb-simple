# dev-haskell-cli-gradle-influxdb-simple

## Description
Creates a small database table
called `dog` and populates with
hql.

## Tech stack
- haskell
- cabal

## Docker stack
- haskell
- influxdb:2

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
